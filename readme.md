# Ethereum Digital Signature Verification

A React-based decentralized application that demonstrates
secure message signing and verification using MetaMask
and Ethers.js.

## Features
- Sign messages using MetaMask
- Verify ECDSA signatures
- Recover Ethereum address from signature
- No backend, no private key storage

## Tech Stack
- React (Create React App)
- Ethers.js
- MetaMask
- JavaScript

## Getting Started

```bash
npm install --legacy-peer-deps
npm start
